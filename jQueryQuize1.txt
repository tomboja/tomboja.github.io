1. In jQuery terminology, what is the difference between the “jQuery function” and 
  the “jQuery object”?

ANS: The dollar sign with the parentheses is the jQuery function. 
     The element that is selected by the jQuery selector is jQuery object

2. Write jQuery code to find all h1 elements that are children of a div element 
  and make their background color red.
Sample HTML:
  <body>
      <h1>abc</h1><br>
      <div>
        <h1>div-1</h1>
        <h1>div-2</h1>
      </div>
      <h1>xyz</h1>
  </body>

  ANS: $('div h1').css('background-color', 'blue');

3. Use a jQuery method to insert the text "YES!" at the end of the <p> element.
<!DOCTYPE html>
<html>
    <head>
        <script>
        <!—INSERT YOUR JQUERY CODE HERE - - >
        </script>
    </head>
    <body>
        <p>Is jQuery FUN or what? </p>
    </body>
</html>

ANS: $('p').append('YES!');
